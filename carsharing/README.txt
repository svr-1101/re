Как запустить проект:

1. Распаковать архив в папку C:\xampp\htdocs\carsharing
2. Запустить XAMPP Control Panel → запустить Apache и MySQL
3. Открыть в браузере http://localhost/phpmyadmin
4. Создать новую базу данных с именем carsharing (кодировка utf8mb4_general_ci)
5. В этой базе перейти на вкладку "Импорт" → выбрать файл carsharing.sql → нажать "Импорт"
6. Открыть в браузере http://localhost/carsharing/index.php

Логин/пароль MySQL: root / (пустой)

Функционал:
- Главная страница с 20 машинами и фото
- Регистрация и вход
- Аренда автомобиля
- Завершение аренды
- Оплата поездки
- Оставление отзыва
- Страница "Мои поездки"
